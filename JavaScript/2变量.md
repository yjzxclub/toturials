# 变量

**变量**本质上是值（例如数字或字符串）的容器，比如你可以用它去存一个数字 `123`，也可以存一个 `Hello world!` 字符串。

变量可以通过这三个关键字声明：

- `let`: 这是在 ECMAScript 2015 里面添加的。它声明一个块作用域的局部变量，可选初始化一个值。
